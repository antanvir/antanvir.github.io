Projects:

- Social Media Response Based Vending Machine
This software collects responses from Twitter and Instagram with a specific, pre-defined hashtag. Shows each response in the display for a few seconds. In the meantime this application emits signal for arduino based vending machine.
This application is built in Python language.
- AnnoTag: Multi-user Data Labelling Tool
A web app to help in text annotation (Labelling). One can create project, upload files, add editors for the project, download labelled file(s). Each editor can label any file of the project, more than one users can work on a single file simultaneously. Editors can also check their performance statistics.
Backend of the application has been built in NodeJs and HTML, CSS, Bootstrap have been used to build the frontend.
